Research and analysis about ‘Slack’ team communication application

This small report includes analysis about Slack which is one of the most popular team communication applications in the market. So, the users who use Slack are actually happy with its flexibility and easiness to use, and you can mostly find pros of the app while investigating the user reviews and comments. However, some issues noticed by the users which would be nice if improved or added:

  • Need to improve the platform, because it is historically linear and the history gets lost easily
  • Needs a rich text editor (can’t add bullet points)
  • You need paid version if going to use the ability to load past conversations and posts
  • Would be better to be able to mark message as unread and to respond to earlier messages in a channels
  • Doesn’t work on previous Windows versions
  • Price
  • For most of the settings for the account, a new tab opens instead of doing everything in the same window
  • There is no feature for Skype calls
  • You cannot create a temporary chat with one or two members, you need to create a new private channel
  • Would be nice if calendar and tasks functions are added
  • There is no video chat
  • It would be good to be able to use GIF reactions
