Rationale
  1. The social need is identified by people becoming less social as remote working is increasing and people are therefore not around other people as much. This is an important skill as almost all projects are based on teamwork where you will need to collaborate. The social skill is very important here in order to create the best possible result.
  2. The need of remote working is identified as it requires travel expenses, cost in having offices, electricity and heating.
  3. The need for application work on all platforms is identified because it would be better if the application can be installed on all platforms(IOS, Android), so that it can be used by everyone no need to change the communication device or the platform they have.
  4. The need to have a to do list in the application is identified because it is obvious to that in a team each member is given a task or the list of the things to do. To have this kind of list will help team members get a balanced allocation of tasks for each employee, and feel more accountable.
