Prioritation list

Prioritation 1

  • The system shall allow users to navigate between different pages

  • The system shall provide users with messaging

  • The system shall provide users with news feed

  • The system shall ask users to include their mood status when they open the application

  • The system shall allow users to send emojis

  • The system shall allow users to see all the teams they are in

  • The system shall allow users to create a new team

  • The system shall allow users to add members to a team

  • The system shall add a member for everyone when someone adds a contact to a group

  • The system shall allow admins of groups to remove members

  • The system shall allow users to pick one or more people to chat

  • The system shall have a page to let users write their status

  • The system shall allow users to share their location

Prioritation 2

  • The system shall allow users to import contacts from ATOS or Facebook

  • The system shall allow the users to tick the boxes in front of the tasks when they are done

  • The system shall provide users with reminders

  • The system shall use RAG status to to let users see their progress

  • The system shall let users to add tasks for the other users

  • The system shall allow users to send pictures and files

Prioritation 3

  • The system shall let users to send standard phrases instead of messages

  • The system shall let users to create groups according to their interest

  • The system shall allow users to send pictures and files

  • The system shall have a profile page for each user

  • The system shall allow users to choose an online or offline status
